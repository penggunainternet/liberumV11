Tabel 4.1 Hasil Pengujian Fungsionalitas API Backend

| No  | Modul             | Skenario Pengujian                                                                       | Hasil yang Diharapkan                                           | Deskripsi                                                                                                     |
| --- | ----------------- | ---------------------------------------------------------------------------------------- | --------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| 1.  | Login Pengguna    | Request: POST ke http://localhost:3000/api/auth/login                                    | Berhasil login, menerima token JWT dalam respons, status 200 OK | Login berhasil. Token JWT diterima di respons body untuk digunakan pada permintaan yang dilindungi.           |
|     |                   | Body (JSON): {"email": "user@example.com", "password": "password123"}                    |                                                                 | Status: 200 OK                                                                                                |
|     |                   |                                                                                          |                                                                 | Respons Body (JSON): {"message": "Login successful", "accessToken": "eyJ0...", "user": {"name": "User Name"}} |
| 2.  | Register Pengguna | Request: POST ke http://localhost:3000/api/auth/register                                 | Berhasil registrasi, akun baru dibuat, status 201 Created       | Registrasi berhasil. Akun baru dibuat dalam sistem.                                                           |
|     |                   | Body (JSON): {"name": "New User", "email": "new@example.com", "password": "password123"} |                                                                 | Status: 201 Created                                                                                           |
|     |                   |                                                                                          |                                                                 | Respons Body (JSON): {"message": "Registration successful", "user": {"name": "New User"}}                     |
| 3.  | Buat Thread       | Request: POST ke http://localhost:3000/api/threads                                       | Thread baru berhasil dibuat dalam database                      | Pembuatan thread berhasil.                                                                                    |
|     |                   | Headers: Authorization: Bearer {token}                                                   |                                                                 | Status: 201 Created                                                                                           |
|     |                   | Body (JSON): {"title": "Judul Thread", "body": "Isi thread", "category_id": 1}           |                                                                 | Respons Body (JSON): {"message": "Thread created", "thread": {"title": "Judul Thread"}}                       |
| 4.  | Update Thread     | Request: PUT ke http://localhost:3000/api/threads/{id}                                   | Thread berhasil diperbarui dalam database                       | Update thread berhasil.                                                                                       |
|     |                   | Headers: Authorization: Bearer {token}                                                   |                                                                 | Status: 200 OK                                                                                                |
|     |                   | Body (JSON): {"title": "Judul Baru", "body": "Isi baru"}                                 |                                                                 | Respons Body (JSON): {"message": "Thread updated", "thread": {"title": "Judul Baru"}}                         |
| 5.  | Hapus Thread      | Request: DELETE ke http://localhost:3000/api/threads/{id}                                | Thread berhasil dihapus dari database                           | Penghapusan thread berhasil.                                                                                  |
|     |                   | Headers: Authorization: Bearer {token}                                                   |                                                                 | Status: 200 OK                                                                                                |
|     |                   |                                                                                          |                                                                 | Respons Body (JSON): {"message": "Thread deleted successfully"}                                               |
| 6.  | Buat Reply        | Request: POST ke http://localhost:3000/api/reply                                         | Reply berhasil dibuat dalam database                            | Pembuatan reply berhasil.                                                                                     |
|     |                   | Headers: Authorization: Bearer {token}                                                   |                                                                 | Status: 201 Created                                                                                           |
|     |                   | Body (JSON): {"thread_id": 1, "body": "Isi komentar"}                                    |                                                                 | Respons Body (JSON): {"message": "Reply created", "reply": {"body": "Isi komentar"}}                          |
| 7.  | List Categories   | Request: GET ke http://localhost:3000/api/categories                                     | Daftar kategori berhasil diambil                                | Pengambilan daftar kategori berhasil.                                                                         |
|     |                   | Headers: Authorization: Bearer {token}                                                   |                                                                 | Status: 200 OK                                                                                                |
|     |                   |                                                                                          |                                                                 | Respons Body (JSON): {"data": [{"name": "Category 1"}, {"name": "Category 2"}]}                               |
